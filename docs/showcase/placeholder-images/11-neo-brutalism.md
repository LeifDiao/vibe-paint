# Neo Brutalism Preview Images

This file lists all image placeholders used in the 11-neo-brutalism.html preview. All images should be generated in bold brutalist style with thick black outlines, hard shadows, high contrast, no gradients, and raw, unpolished aesthetics.

## Image Specifications

**Style Guidelines:**
- Bold brutalist style with thick black outlines (3px minimum)
- Hard drop shadows (4px 4px 0 black) with no blur
- High contrast colors: bright primaries (#FF6B6B, #4ECDC4, #A8E6CF, #FFE66D) against near-black (#1A1A2E) or white
- No gradients, no blur, no rounded corners or minimal (2px max)
- Raw, unpolished aesthetic with visible borders and stark design
- Space Grotesk typography when text is included, bold and heavy weights
- 1920x1080px or appropriate aspect ratio for placement

## Color Palette for All Images

## Design System Token References

- **Border:** 3px solid #1A1A2E (minimum)
- **Shadow:** 4px 4px 0 #1A1A2E (no blur)
- **Hover Shadow:** 2px 2px 0 #1A1A2E
- **Font:** Space Grotesk, bold weights (600, 700)
- **Border Radius:** 0px (sharp corners) or 2px maximum

## Notes

- All images should maintain the raw, industrial aesthetic of Neo Brutalism
- No smooth gradients, no anti-aliasing softness
- Visible borders and hard shadows define depth
- High contrast ensures accessibility and visual impact
- Typography should be bold, uppercase when appropriate, and use Space Grotesk or similarly heavy geometric fonts
