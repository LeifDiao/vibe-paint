# Candy Pop - Placeholder Images Guide

This document describes placeholder image filenames and generation prompts for the Candy Pop design system.

## Image Generation Style Guide

All images should follow this aesthetic:
- **Style**: Candy pop aesthetic, bright pastels, hot pink and lavender, cute kawaii feel
- **Colors**: Hot pink (#FF69B4), Lavender (#B19CD9), Mint Green (#98FB98), light pastels
- **Shapes**: Rounded bubbly shapes, playful and fun, no sharp edges
- **Mood**: Cheerful, playful, whimsical, joyful, delightful

## Notes for Implementation

1. **Fallback Gradient**: If images fail to load, CSS provides pastel gradient fallbacks in hot pink and lavender
2. **Formats**: Recommend PNG or WebP for transparency support
3. **Optimization**: Compress images to reduce file size while maintaining quality
4. **Style Consistency**: All images should maintain the playful, bubbly, pastel aesthetic
5. **Brand Colors**: Primary (#FF69B4), Secondary (#B19CD9), Tertiary (#98FB98)

## Generation Tools

These images can be created using:
- AI image generation (Midjourney, DALL-E, Stable Diffusion)
- Design software (Figma, Adobe Creative Suite)
- Illustration tools (Procreate, Adobe Fresco)
- Vector design (Illustrator, Inkscape)

Use the prompts above and ensure consistent styling across all images.
